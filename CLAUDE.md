# Project rules

This is Anja's live business website (naturallywithanja.com), hosted on GitHub Pages.

- Only run `git commit` and `git push` when Anja explicitly approves it (e.g. "push this," "make it live," "yes go ahead"). Never commit or push automatically after making edits.
- Make and explain edits first, then wait for approval before pushing.
- The custom domain is configured via the `CNAME` file — do not delete or modify it.
